# Where to paste Git commands

Paste the commands in a **terminal** (Command Prompt, PowerShell, Terminal app, or VS Code Terminal) that is opened in this repository folder.

## Quick steps
1. Open a terminal.
2. Go to this repo folder:
   ```bash
   cd /workspace/Literature
   ```
3. Paste and run commands one line at a time, for example:
   ```bash
   git branch --show-current
   git remote -v
   git push -u origin work
   ```

If you use GitHub Desktop, use **Repository → Open in Terminal** first, then paste commands there.
